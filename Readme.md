# AI Robotic Task Planner

AI-powered robotic task planning system using Qwen 2.5, Ollama, Flask, and PyBullet.

## Problem Statement
Traditional robotic systems rely on predefined and hardcoded instructions, limiting their flexibility in understanding natural human commands and adapting to task variations.

This project aims to develop an AI-powered robotic task planning system that enables a robot to understand natural language instructions, convert them into structured executable actions using a Large Language Model (Qwen 2.5), and simulate robotic manipulation through a physics-based environment using PyBullet.
